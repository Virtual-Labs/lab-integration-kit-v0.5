1 Introduction

This document refers to web application security toolkit. The security toolkit explains secure web development practices for developing virtual labs. It also contains wapiti a black-box web application vulnerability scanner. Developers are advised to use this toolkit for developing secure virtual labs and also test the security of the virtual lab.

2 Toolkit Structure

This toolkit follows the following structure 

%-- docs # Contains all of the documentation and tutorials.
%   %-- README.html # This readme
